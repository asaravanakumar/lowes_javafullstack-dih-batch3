STEP 1: Download Postgres for Windows - https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
STEP 2: Run postgresql-15.x-windows-x64.exe as Adminstrator (follow the wizard - provide superuser credentials [postgres\postgres])
STEP 3: Open Command Prompt and Navigate to %POSTGRES_HOME%\bin e.g:> cd "C:\Program Files\PostgrSQL\15\bin"
STEP 4: Run PostgreSQL shell client :> psql
STEP 5: Provide the details (host, port, database, username, password) to connect to PostgreSQL server
